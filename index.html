GitHub

How to use it?

A Guide!

Git Installation

First step is to install the git setup in your system and configure it. Git can be downloaded from This Link.

Initialize Git

Open the folder you want to work in and initialize it with "git init" command.

Check Status

Check the status of added/modified files in the folder with "git status" command.

Add the changes

Add the changes made to local repo using "git add ." command. "." will add all required files.

Commit the changes

Commit the changes made to local repo using (git commit -m "Message") command.

Repo on GitHub

Now create a online repository on Git hub

Link the Github with git.

Link your git and gitHub repositories using "git remote add origin (copy address from github)"

Push request

Push the local files to remote repository using "git push command" and "git push --set--upstream origin" if using push for the first time.

Pull request

You can fetch and download content from a remote repository and update the local repository using "git pull" command.

Clone the remote repository

Clone the remote repository to your local system using "git clone Address"

Branching

To avoid conflict when multiple people working on same project git provides branching.There's one master branch and then can be mulitple branches. You can make a branch using "git branch feature1" and then merge the branches using "git merge branchname".
